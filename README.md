# Portfolio Website

A static personal portfolio website ready for GitHub Pages.

## Edit Content

- Update `index.html` with your name, bio, project titles, project links, and contact links.
- Update `styles.css` to adjust colors and layout.
- Replace placeholder project blocks with screenshots when you have final assets.

## Publish With GitHub Pages

After pushing this repository to GitHub:

1. Open the repository on GitHub.
2. Go to **Settings** > **Pages**.
3. Under **Build and deployment**, choose **Deploy from a branch**.
4. Select the `master` branch and `/root`, then save.

Your site will be available at the GitHub Pages URL shown in that settings page.
